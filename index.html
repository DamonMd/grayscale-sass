Travel Planner

Allows users to create a trip then search for things to do on that trip. Option to sort activities by day using a drag and drop feature. Created using MongoDB/Mongoose, Express.js, Handlebars.js, jQuery and Node.js with the Google Places API

RevYou

Envisioned as an app that would bring you reviews for activities in a given city from profiles similar to your Facebook or Twitter profile. Ruby on Rails app featuring O-Auth Facebook and Twitter API, Google Maps JS API, YELP Ruby API, and PostgreSQL database.

See RevYou on GitHub
