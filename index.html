Grayscale-sass

The SASS version of StartBootstrap's Grayscale template

Download this project as a .zip file Download this project as a tar.gz file

I do not take credit, for the designing of this template!

The original version can be found at Start Bootstrap - Grayscale

Grayscale is a multipurpose, one page HTML theme for Bootstrap created by Start Bootstrap. This template features various content sections and a Google Maps section with a custom map marker.

Getting Started

To use this template, choose one of the following options to get started: 0. (optional) Install bower, compass, sass 1. Fork/download this repository on GitHub 2. Install bower dependencies (bower update) 3. Compile SASS (compass compile)
